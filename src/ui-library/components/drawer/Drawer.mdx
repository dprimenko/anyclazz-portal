# Drawer

Drawer component with different props.

#### Example

<Canvas />

## Usage

```ts
import { Drawer } from "@docline/ui";

const Example = () => {
  return (
      <Drawer>
        {children}
      </Drawer>
  );
};

export default Example;
```

#### Arguments

- **`children`** - (JSX) Children element
- **`width`** - (number) Width in %

- **`data-testid`** _(type: `string`)_
A string identifier used for testing purposes. It allows the component to be easily selected in testing frameworks.
