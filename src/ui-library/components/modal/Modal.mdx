# Modal

Modal component with different props.

#### Example

<Canvas />

## Usage

```ts
import { Modal } from "@docline/ui";

const Example = () => {
  return (
      <Modal 
        title="Modal"
        onClose={() => ()}
        persistent
        actions={
          <>
            <Button label="Cancel" />
            <Button label="Confirm" />
          </>
        }
      >
        {children}
      </Modal>
  );
};

export default Example;
```

#### Arguments

- **`children`** - (JSX) Children element
- **`title`** - (String) Modal title
- **`onClose`** - (Function()) onClose handler
- **`actions`** - (JSX) Actions slot (Bottom area)
- **`persistent`** - (Boolean) onClose not trigger on outside click
- **`width`** - (Number) Width of modal in PX

- **`data-testid`** _(type: `string`)_
A string identifier used for testing purposes. It allows the component to be easily selected in testing frameworks.
