# Dropdown

Dropdown component with different props.

#### Example

<Canvas />

## Usage

```ts
import { Dropdown } from "@docline/ui";

const Example = () => {
  return (
      <Dropdown
         label="Introduce tu nombre"
         rightIcon="<X />"
         onClick={() => ()}
      />
  );
};

export default Example;
```

#### Arguments

- **`items`** - (DropdownOptionProps) Dropdown options
- **`fullWidth`** - (Boolean) Dropdown fill available X space
- **`variant`** - (String) Dropdown variant (big, small)
- **`value`** - (String) Dropdown value
- **`label`** - (String) Dropdown input
- **`required`** - (Boolean) Is input required?
- **`error`** - (Boolean) Dropdown state error enabled
- **`errorMessage`** - (String) Dropdown error message
- **`noErrorMessage`** - (Boolean) Hide error message slot
- **`$disabled`** - (Boolean) Disables input

- **`data-testid`** _(type: `string`)_
A string identifier used for testing purposes. It allows the component to be easily selected in testing frameworks.
