# Chip

Chip component with different props.

#### Example

<Canvas />

## Usage

```ts
import { Chip } from "@docline/ui";

const Example = () => {
  return (
      <Chip
         label="Chip"
         $color="#10587E"
         $bgColor="#EDF4FF"
         $hasBorder
      />
  );
};

export default Example;
```

#### Arguments

- **`label`** - (String) Chip text
- **`$color`** - (String) Chip text color
- **`$bgColor`** - (String) Chip background color
- **`$hasBorder`** - (Boolean) Add border to chip container

- **`data-testid`** _(type: `string`)_
A string identifier used for testing purposes. It allows the component to be easily selected in testing frameworks.
